# 🌿 Saquarema Verde - Backend & Admin Panel

Repositório destinado ao backend e ao painel administrativo (versão web) do projeto **Saquarema Verde**.

## 📌 Descrição do Projeto
O Saquarema Verde é uma plataforma para catalogar e divulgar eventos, trilhas, praias e pontos turísticos da região de Saquarema. Este repositório contém a API (Node.js/Express) que alimenta o aplicativo mobile e o sistema web de administração de conteúdo.

## 🚀 Tecnologias Utilizadas
- **Node.js** com **Express** (Gerenciamento de Rotas e API)
- **MySQL** (Banco de Dados Local)
- **JWT (JSON Web Tokens)** (Autenticação e Segurança)
- **Bcryptjs** (Criptografia de senhas)
- **Multer** (Upload de arquivos e vídeos locais)
- **HTML/CSS/JS (Vanilla)** (Painel Web Administrativo em `/public`)

## ⚙️ Requisitos para Rodar o Projeto
- **XAMPP** ou **WAMP** instalados para rodar o servidor MySQL local.
- **Node.js** (versão 18+ recomendada) e npm.
- Celular e computador devem estar na mesma rede Wi-Fi para testar a comunicação com o app mobile.

## 🛠️ Como Executar Localmente
1. Inicie o Apache e o MySQL pelo painel do XAMPP/WAMP.
2. Crie um banco de dados no phpMyAdmin chamado `saquarema_verdee`.
3. Clone este repositório e abra a pasta no terminal.
4. Instale as dependências:
   ```bash
   npm install
   ```
5. Inicie o servidor:
   ```bash
   npm start
   ```
   *(O servidor rodará na porta 8080)*

6. **Acesso:**
   - **Painel Admin:** `http://localhost:8080/admin.html`
   - **Página Inicial Web:** `http://localhost:8080/index.html`
