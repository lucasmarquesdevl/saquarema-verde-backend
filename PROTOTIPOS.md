# 💻 Telas e Funcionalidades (Painel Web)

Este documento apresenta as telas do Painel Administrativo Web e da Página Inicial (Frontend) do projeto **Saquarema Verde**.

---

### 1. Página Inicial Pública (Feed)
Visão que o usuário comum tem ao acessar o site pelo navegador. Mostra a listagem de eventos com o reprodutor de vídeo.
* **Destaques:** Listagem de praias, trilhas e pontos turísticos.
* **Acesso:** Botão no topo para a área do administrador.

![Página Inicial](./public/images/tela_index.jpg)

---

### 2. Tela de Login (Acesso Admin)
Página web restrita aos administradores.
* **Segurança:** Autenticação via JWT.

![Tela de Login](./public/images/tela_login_web.jpg)

---

### 3. Painel Administrativo
Tela de gerenciamento onde o administrador pode inserir, editar e excluir eventos, além de fazer o upload do vídeo promocional no formato MP4.
* **Formulário:** Inserção de dados de texto e botão para anexar mídia.
* **Listagem:** Visualização em tempo real de tudo que está salvo no banco de dados.

![Painel Admin](./public/images/tela_admin_web.jpg)
